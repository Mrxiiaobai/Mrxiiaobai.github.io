2、客户端

2.1、安装

socket.io v4.x版本 至少需要 Nodev10

2.2、下载

npm install socket.io-client

2.3、使用

通过client_online自定义事件，向服务端传输当前用户信息，服务端保存当前连接数量及用户

通过client_msg自定义事件，向服务端发送聊天信息

通过监听server_msg，获取服务端返回的消息，并保存
