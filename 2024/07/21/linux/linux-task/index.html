1、linux自带crontab、没有需要安装

crontab -l 查看定时任务
crontab -e 编辑定时任务
每五分钟执行 */5 * * * *

每小时执行 0 * * * *

每天执行 0 0 * * *

每周执行 0 0 * * 0

每月执行 0 0 1 * *

每年执行 0 0 1 1 *
